# Sign Language to Voice/Text Conversion

## Project Overview

The Sign Language to Voice/Text Conversion System is an AI-powered communication solution designed to bridge the gap between sign language users and non-sign language users. The system captures hand gestures using a camera, processes them through computer vision and machine learning techniques, and converts them into understandable text and speech output in real time.

## Problem Statement

Individuals who rely on sign language often face communication challenges when interacting with people who do not understand sign language. Existing solutions are either expensive, require specialized hardware, or do not provide real-time gesture-to-voice conversion.

## Proposed Solution

This project uses a camera-based gesture recognition system to detect hand signs and translate them into text and speech. The system aims to provide an affordable, accessible, and user-friendly communication tool for deaf and speech-impaired individuals.

## Key Features

* Real-time hand gesture detection
* Camera-based sign recognition
* Text output generation
* Voice output using Text-to-Speech
* User-friendly interface
* Low-cost implementation
* No specialized hardware required

## Technology Stack

* HTML
* CSS
* JavaScript
* MediaPipe Hands
* Computer Vision
* Machine Learning Concepts
* Web Camera Interface

## System Workflow

1. Capture hand gesture through camera.
2. Extract hand landmarks using MediaPipe.
3. Process gesture data.
4. Recognize the corresponding sign.
5. Convert the recognized sign into text.
6. Generate speech output using Text-to-Speech.
